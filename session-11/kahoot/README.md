5. Como acessar a propriedade `color`?

```js
const car = {
  brand: "Mercedes Benz",
  model: "Mercedes-AMG GLB 35 4MATIC",
  color: "Deep Blue",
}
```

- ✅ `car.color`
- ❌ `./car/color`
- ❌ `car.filter(elem => elem === "color")`
- ❌ `car(color)`

6. Qual outra forma de acessar a propriedade `color`?

- ❌ `car[color]`
- ❌ `./car/color`
- ❌ `car.filter(elem => elem === "color")`
- ✅ `car["color"]`
